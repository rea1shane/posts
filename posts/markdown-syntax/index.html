Heading 标题

下面的 HTML H1 - H6 元素代表六个级别的章节标题。H1 是最高的章节级别，H6 是最低的，语法：

# H1

## H2

### H3

#### H4

##### H5

###### H6

效果：

H1

H2

H3

H4

H5
H6

Paragraph 段落

语法：

不同的段落之间需要有一行空行。

如果没有空行的话会被视为同一段落
回车符会被视为空格

效果：

不同的段落之间需要有一行空行。

如果没有空行的话会被视为同一段落 回车符会被视为空格

Blockquotes 块引用

块引用表示从其他来源引用的内容，语法：

> > > 这是一段描述。
> >
> > 这点还不错！
> >
> > > 这是另一段描述。
> >
> > 这个就差点意思...
>
> 这些都是引用。

效果：

这是一段描述。

这点还不错！

这是另一段描述。

这个就差点意思…

这些都是引用。

脚注

语法：

这是一个脚注。[^1]

[^1]: 这是脚注的内容。

这是另一个注脚。[^2]

[^2]: 这是另一个脚注的内容。

效果：

这是一个脚注。1

这是另一个注脚。2

Tables 表格

可以使用冒号来定义表格的对齐方式，语法：

| Name  | Age |   Skin | Describe |
|:----- |:---:| ------:| -------- |
| Bob   | 27  |  white | tall     |
| Alice | 23  | yellow | fat      |

效果：

NameAgeSkinDescribe
Bob27whitetall
Alice23yellowfat

Code block 代码块

backticks 通过反引号声明

语法：

```
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8" />
        <title>Example HTML5 Document</title>
    </head>
    <body>
        <p>Test</p>
    </body>
</html>
```

效果：

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8" />
        <title>Example HTML5 Document</title>
    </head>
    <body>
        <p>Test</p>
    </body>
</html>

可以在前面的三个反引号后声明代码的语言种类，语法：

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8" />
        <title>Example HTML5 Document</title>
        <meta name="description" content="Sample article showcasing basic Markdown syntax and formatting for HTML elements.">
    </head>
    <body>
        <p>Test</p>
    </body>
</html>
```

效果：

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8" />
        <title>Example HTML5 Document</title>
        <meta name="description" content="Sample article showcasing basic Markdown syntax and formatting for HTML elements.">
    </head>
    <body>
        <p>Test</p>
    </body>
</html>

four spaces 通过四个空格声明

语法：

    <!doctype html>
    <html lang="en">
    <head>
      <meta charset="utf-8">
      <title>Example HTML5 Document</title>
    </head>
    <body>
      <p>Test</p>
    </body>
    </html>

效果：

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>

List Types 列表

Ordered List 有序列表

语法：

1.  First item
2.  Second item
3.  Third item

或者：

1.  First item
1.  Second item
1.  Third item

效果：

  1. First item
  2. Second item
  3. Third item

Nested list 无序列表

语法：

-   Fruit
    -   Apple
    -   Orange
    -   Banana
-   Dairy
    -   Milk
    -   Cheese

效果：

  • Fruit
    • Apple
    • Orange
    • Banana
  • Dairy
    • Milk
    • Cheese
